CONTRACT AUTO-GENERATION FROM TRANSACTIONS

Overview:
  Automatically generate rental contracts from transaction records using a professional agency template. The feature should streamline rental agreement creation and preserve brand identity.

Acceptance Criteria:
  ✓ Agent can select any transaction and click "Generate Contract"
  ✓ User is redirected to a contract preview page
  ✓ Contract is pre-filled with transaction and property data
  ✓ Agency logo appears at the top of the contract
  ✓ Contract follows a clean, professional rental agreement design
  ✓ Sections can be edited, removed, or added before finalizing
  ✓ Contract template is based on the sample file at ./Contrat_Location.docx
  ✓ Generated contracts can be exported or downloaded
  ✓ Draft versions are saved for later editing

Technical Notes:
  - Add a contract generation service and storage for generated documents
  - Use a template engine or document library for DOCX/PDF generation
  - Save generated contract metadata to the transaction record
  - Provide UI for contract editing and preview

Priority: Medium
Dependencies: Transaction module, agency branding settings
