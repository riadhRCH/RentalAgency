# 03 - Transactions & Document Management

## 3.1 Contract Generation Refinements
**Status:** Pending
**Priority:** High
**Scope:**
*   **Branding:** Contracts should display the platform's logo/name by default, rather than the agency account (or clarify white-labeling rules).
*   **Data Accuracy:** Fix bugs related to payment frequency and lease duration (durée).
*   **Templates:** Differentiate base document templates for Short-Term vs. Long-Term rentals.
*   **Export:** Implement proper PDF download instead of HTML format.
*   **Signatures:** Add ability to insert a signature image (photo).
*   **Sharing:** Allow sharing the generated contract via a secure link.

## 3.2 Closed Transaction Client Portal
**Status:** Pending
**Priority:** Medium
**Scope:**
*   Design a post-closure experience: extract a secure link for closed transactions.
*   Allow the tenant/buyer to view their final contract.
*   Add a feature for the user to submit a "Welcome Selfie" (Selfie de bienvenu).
*   Include a form/input for the user to submit claims or maintenance requests (réclamations).
